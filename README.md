## 碎碎念
趁着春节回家，壮志踌躇的希望用四五天的功夫去开发一个简单的诗词APP。现在诗词APP有很多，像古文岛、西窗烛之类的很多APP都非常优秀。

既然如此，为什么还要单独开发一款新的APP呢？

原因很简单，就是付费。这些软件都很好，但是如果你希望去看某个诗人的作品，或者想要看注释或者赏析，那么就需要去内购一些会员功能。我既不愿意登录，更不愿意付费，于是我开发了这款APP。

诗词的注释和赏析确实需要很大的成本，因为这些内容都是许多老师和学者的创作成果，我们要尊重他们的劳动成本，而这部分人力成本是很难省下来的。

**既然人力成本很贵，那么我们就不用人力了。** 在口袋诗词中，我们用AI来实现这些功能，甚至可以做到”哪里不会点哪里“的划线提问。你所需要的仅仅是购买一定API额度（这个很便宜，甚至阿里、字节等API提供商会有很高的免费额度）。

> 不要问为什么我不提供额度，咱连服务器的成本都剩下了，能给你免费的AI调用额度？？

在这个年代，信息有时比软件更加珍贵，感谢github的开源项目 [chinese-poetry](https://github.com/chinese-poetry/chinese-poetry)，提供了数据源，让我的这个项目成为了可能。（如果数据有问题不要找我啊喂！）

以及令人意想不到的是这个APP的开发过程，本来我以为有AI的辅助，几天时间能够开发出一个APP了。毕竟我作为pilot，AI做我的copilot，我们双剑合璧打遍天下无敌手。但谁曾想，最终的画风是AI是一个driver，我是那个坐在副驾驶上指指点点的乘客，只需要指出我的目的地，以及不停的进行抱怨。（嗯，还是做领导好啊 x）

## 功能
- 读诗：首页的每日一句以及【诗库】和【诗库】的搜索，是获取诗词的来源
- 收藏：点击诗词卡片进入详情页，会看到收藏按钮，收藏的诗词会展示在【辑录】页面
- AI辅助：在设置页配置您的AI调用Key，然后就可以使用诗词详情页的【AI解释】和【AI赏析】了。同时，您还可以长按选中文本进行搜索。
